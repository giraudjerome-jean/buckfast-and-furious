BUCKFAST AND FURIOUS

What this package contains:
- A working front-end site (index.html)
- A live data file (data/archive.json)
- A ready daily automation file for GitHub Actions
- A placeholder fetch script to be connected to your source retrieval

Important:
A website cannot autonomously fetch a new article every day from a local HTML file alone.
It needs a scheduler/server service.
This package is therefore ready for GitHub upload and daily automation, but the source-retrieval step must be connected.
